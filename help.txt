
Per compilar en mode desenvolupament: deploydev.sh/bat [OPCIONS]
Per compilar en mode producció: deploypro.sh/bat [OPCIONS]

-------------------------------- OPCIONS ---------------------------------

-Pbuild           Nomes emprar en desenvolupament. Sistema manual de 
                  revisions. Marca una nova data de revisió.

-Psqlgen          Genera scripts SQL dins /scripts/sqlgenerator. Mes
                  informacio a /scripts/sqlgenerator/readme.txt

-Pminiappletui    Compila l'applet i el firma amb el afirma.keystore. Sino
                  s'ha d'executar install_portafib_applet_signed_jar.sh/bat

-Psia             Compila i inclou dins l'ear el Modul de Firma de SIA

-Ppassarelaweb-v1    Compila el Modul de Firma Web de Passarel·la PortaFIB
                  Tambe crea l'API i WS per atacar a Passarela Web de PortaFIB.
                  Els WS definits son 
                       /portafib/ws/v1/PortaFIBPassarelaDeFirmaWeb

-Ppassarelaserver-v1 Compila el Modul de Firma en Servidor de Passarel·la PortaFIB
                  Tambe crea l'API i WS per atacar a Passarela Server de PortaFIB.
                  Els WS definits son 
                       /portafib/ws/v1/PortaFIBPassarelaDeFirmaEnServidor

-Pclientcert      Genera un segon context web amb autenticacio
                  CLIENT-CERT (context a /portafib/s)

-Pws-portafib-v1  Crea WS per atacar a PortaFIB usant v1. Els WS definits son 
                       /portafib/ws/v1/PortaFIBPeticioDeFirma
                       /portafib/ws/v1/PortaFIBUsuariEntitat
                       /portafib/ws/v1/PortaFIBUsuariAplicacio             
                  Exemples d'us a ws\portafib_api i ws\portafib_api_example.

-Pws-portafib-callback-server  Crea un conjunt de serveis per testejar la 
                  funcionalitat de callback de PortaFIB:
                  - WS per provar la recepcio dels Callback de PortaFIB via WS.
                    Els usuaris aplicacio han de tenir callback a una url com
                    http://HOST:PORT/portafib/cb/v1/PortaFIBCallBack i el
                    camp 'Callback Versio" a 'PortaFIB Callback WS v1.0'
                  - REST per provar la recepcio dels Callback de PortaFIB via REST.
                    Els usuaris aplicacio han de tenir callback a una url com
                    http://HOST:PORT/portafib/cbrest/v1/event (POST)
                    i camp 'Callback Versio" a 'PortaFIB Callback REST v1.0'
                  - WEB: UI per poder consultar els events de Callback enviats
                    via WS o via REST. La url és http://HOST:PORT/portafib/portafibcb

-Pws-portafirmas  Crea WS compatible amb l'API del PortaFirmas antic de la CAIB.
                  El WS es accessible des de /portafib/portafirmasws/web/services/CWS.
                  Exemple de cridada a ws\indra_api.

-Pws-portafirmas-callback-server Crea WS per provar la recepcio dels Callback del
                  PortaFirmas antic de la CAIB. Els usuaris aplicacio han de tenir
                  calback url 
                  http://HOST:8080/portafib/portafirmascb/v0/PortafirmasCallBack
                  i callback versio 0. Per veure les cridades accedir a
                  http://HOST:8080/portafib/portafirmascb

-Pws-portafib-v2  [EN DESENVOLUPAMENT. NOMES PEL FUTUR PORTAFIB v2.0.0]
                  Crea WS per atacar a PortaFIB usant v2. Els WS definits son 
                       /portafib/ws/v2/PortaFIBPeticioDeFirma
                       /portafib/ws/v2/PortaFIBUsuariEntitat
                       /portafib/ws/v2/PortaFIBUsuariAplicacio
                       /portafib/ws/v2/PortaFIBHelloWorld                       
                  Exemples d'us a ws\portafib_api i ws\portafib_api_example.

-Ppassarelaweb-v2 [EN DESENVOLUPAMENT. NOMES PEL FUTUR PORTAFIB v2.0.0]
                  Compila el Modul de Firma Web de Passarel·la PortaFIB
                  Tambe crea l'API i WS per atacar a Passarela Web v2 de
                  PortaFIB. Els WS definits son 
                       /portafib/ws/v2/PortaFIBPassarelaDeFirmaWeb

-Ppassarelaserver-v2 [EN DESENVOLUPAMENT. NOMES PEL FUTUR PORTAFIB v2.0.0]
                 Compila el Modul de Firma en Servidor de Passarel·la PortaFIB
                  Tambe crea l'API i WS per atacar a Passarela Server v2 de
                  PortaFIB. Els WS definits son 
                       /portafib/ws/v2/PortaFIBPassarelaDeFirmaEnServidor

-Dclavefirma      Al ser incompatible el plugin de FIRe amb el de Cl@veFirma,
                  aquest darrer esta desactivat. Aquesta opcio serveix per
                  utilitzar Cl@veFirma enlloc de FIRe.

clean             Neteja els projectes abans de compilar.
